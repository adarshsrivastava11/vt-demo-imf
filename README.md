# How to Run
## Install Django - 
https://www.djangoproject.com/download/
## Install Matplotlib - 
pip install matplotlib
## Install Mpld3 - 
pip install mpld3
## Running the Server - 
```
1. git clone git@github.com:adarshsrivastava11/vt-demo-imf.git
2. cd vt-demo-imf
3. python manage.py runserver

```
